# 🧠 MemoryGame
Игра «Memory» (Найди пару)

## 📝 Описание игры
Приложение реализует классическую игру на запоминание парных карт. Игрок открывает карты по одной, стараясь найти две с одинаковым значением. Цель — найти все пары за минимальное время и количество ходов.

Программа позволяет выбрать уровень сложности, менять тему оформления и ставить игру на паузу.

Проект написан с использованием Windows Forms и принципов объектно-ориентированного программирования.

## ⚙️ Используемые технологии
*   **Платформа:** .NET 8.0
*   **Язык программирования:** C#
*   **Среда разработки:** Visual Studio 2022 и новее
*   **Типы проектов:**
    *   Приложение Windows Forms
    *   Библиотека классов
    *   Тестовый проект MSTest
    *   Инструменты покрытия кода (coverlet, ReportGenerator)

## 🗃️ Структура проекта
*   **MemoryGame (UI)** — пользовательский интерфейс приложения (Windows Forms). Форма с игровым полем и элементами управления.
*   **MemoryLibrary** — основная логика игры. Классы `GameController` (управление игрой) и `MemoryCard` (отдельная карта).
*   **MemoryTests** — модульные тесты для проверки ключевых компонентов библиотеки.

## 📣 Основные возможности
*   Выбор уровня сложности (4, 8 или 12 пар) через выпадающий список.
*   Автоматическая генерация и перемешивание карт.
*   Отображение текущего времени игры и количества сделанных ходов.
*   Таймер обратного переворота несовпавших карт (750 мс).
*   Кнопка «Новая игра» для перезапуска без ожидания победы.
*   **Визуальная подсветка карт** при наведении — подсвечиваются только доступные для выбора карты.
*   **Светлая и тёмная темы** — переключение одной кнопкой, все элементы интерфейса меняют цвета.
*   **Пауза** — останавливает время и блокирует клики.
*   **Умная сетка** — автоматически подбирает число столбцов, минимизируя пустые ячейки (например, для 12 пар — красивая сетка 6×4).
*   Окно автоматически масштабируется под выбранный размер игрового поля.

## 💡 Особенности реализации
*   Чёткое разделение ответственности: библиотека `MemoryLibrary` не зависит от UI.
*   Состояния карт (`IsRevealed`, `IsMatched`) управляются в классе `MemoryCard`.
*   Динамическое создание и удаление кнопок карт при старте игры.
*   Прямое управление цветами без дизайнера.
*   В тёмной теме обеспечена читаемость цифр: на белой открытой карте текст чёрный, на зелёной совпавшей — чёрный.
*   Обработка крайних случаев: защита от повторного клика, блокировка действий во время паузы и анимации переворота.

## ⏳ Тестирование
В проекте **MemoryTests** реализованы модульные тесты на базе MSTest.

Проверяются:
*   Корректность создания нужного количества карт при инициализации.
*   Поведение метода `CheckMatch` при одинаковых и разных `PairId`.
*   Установка свойства `IsMatched` только при совпадении.
*   (Расширяется) Тесты на окончание игры и подсчёт ходов.

## 📊 Покрытие кода
Для замера покрытия используется связка **coverlet** + **ReportGenerator**.

1. Убедись, что установлен генератор отчётов (глобально):
   ```
   dotnet tool install -g dotnet-reportgenerator-globaltool
   ```
2. Запусти скрипт `coverage.cmd.bat` из корня решения.
3. После выполнения в папке `coverage-report` появится HTML-отчёт (откроется автоматически).

## ▶️ Запуск проекта
1.  Откройте `MemoryGameSolution.slnx` в Visual Studio.
2.  Установите проект **MemoryGame** как запускаемый.
3.  Нажмите `Start` (F5).

## ⚠️ Примечание
Для работы требуется **.NET 8.0 SDK** и **Visual Studio 2022** (или новее).
